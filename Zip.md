# Zip Compression of files

```bash
gzip fileaname
```

it compresses it into a .gz file

```bash
gunzip filename.gz
```

uncompresses a \*.gz file

```bash
tar cvf tarfilenane.tar [list of all the files]
```

c means create.

v means verbose.

f means file.

```bash
tar xvf tarfilenane.tar
```

x means Extract.

v means verbose.

f means file.
