# Processes and Jobs

```bash
ps ax
```

The & sends it into the background

```bash
xlogo &
```

```bash
jobs
```

list all the jobs that are running

```bash
fg %1
```

brings a job back to the forground with job no.
indicated with the no after the percent sign

```bash
kill PID
kill -STOP PID
kill -CONT PID
```
